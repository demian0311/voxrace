# Voxel Tank World

A simple 3D voxel world where you can drive a tank, built with Three.js.

## How to Run

Since this project uses ES modules, you need to serve it with a local web server.

### Using Python (Pre-installed on macOS)
1. Open a terminal in this folder.
2. Run:
   ```bash
   python3 -m http.server
   ```
3. Open your browser to `http://localhost:8000`.

### Using Node.js
1. If you have Node.js installed, you can use `npx`:
   ```bash
   npx serve .
   ```

## Controls
- **W / S** or **Arrows**: Drive Forward / Backward
- **A / D** or **Arrows**: Turn Left / Right
- **Space**: Shoot
